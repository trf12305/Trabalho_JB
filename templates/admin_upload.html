Limite: 50 MB · Formato: lista JSON com registros do Siprov

Como gerar o JSON no Siprov

  1. Acesse acesso.siprov.com.br e logue.
  2. Vá em Relatórios → Layout 496.
  3. Filtre o período desejado (ex: Maio/2026).
  4. Selecione todas as situações: LIQUIDADO, ABERTO, PENDENTE.
  5. Exporte como JSON e baixe o arquivo.
  6. Envie aqui — o dashboard atualiza automaticamente.
